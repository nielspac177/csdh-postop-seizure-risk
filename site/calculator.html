How the calculator works

  1. Inputs are z-scaled using BIDMC training-set statistics.
  2. Firth penalized logistic regression returns a probability.
  3. Class-conditional conformal sets are computed at the chosen α.
  4. The recommendation reflects the singleton (or doubleton) prediction.
